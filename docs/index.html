Cutout

Drag-and-drop background removal for macOS, powered by Apple's Vision framework — fully on-device, no uploads.

⬇︎ Download for macOS Free & open-source · Requires macOS 14 or later

Features

  • On-device Apple Vision — nothing uploaded
  • Drag & drop, one image or a whole batch
  • Before / after comparison slider
  • Transparent PNG output
  • JPEG, PNG, HEIC, TIFF, WebP & more
  • Touch-up brush to refine the mask

How to open the app

Cutout is open-source but not signed with a paid Apple Developer ID, so macOS blocks it on first launch. Any one of these opens it (you only need to do this once):

  1. Right-click (or Control-click) Cutout in your Applications folder → Open → Open.
  2. If that's blocked on newer macOS: open System Settings → Privacy & Security, scroll down, and click "Open Anyway".
  3. Or, from Terminal: 
    xattr -dr com.apple.quarantine /Applications/Cutout.app
